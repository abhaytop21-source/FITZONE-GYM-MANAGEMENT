# 🏋️ FITZONE Gym Management System

FITZONE is a modern **Gym Management System** developed using **HTML5, CSS3, and JavaScript**. It provides a professional website for gym visitors and a dedicated admin dashboard for gym owners to manage members, attendance, and payments.

The project demonstrates responsive web design, client-side data management using Local Storage, and interactive JavaScript functionality without requiring a backend database.

---

# 📌 Project Overview

FITZONE is designed to simplify gym management by providing a centralized dashboard where gym owners can manage members, track attendance, monitor payments, and view important statistics.

The project follows a modular structure, where different pages communicate through JavaScript and Local Storage to simulate a real-world management system.
